## 개요
직장인 익명 커뮤니티 앱. [[character/붕붕카]]가 매 화 **감성 게시물을 올리는 무대**. 시리즈의 **프레임 장치** — 모든 에피소드는 블라인드 게시물 스크린샷으로 시작한다.

## 시각 레퍼런스
- **배경**: 순백색. 다크모드 아님.
- **상단**: 회사명(파란색 "현대자동차") + 닉네임 + 작성 시간
- **본문**: 검은 고딕체. 여백 넉넉.
- **하단 액션바**: 👍 좋아요 / 💬 댓글 / 📤 공유
- **댓글**: 연한 회색 배경 블록. 회사명 파란색 강조.

## 연출 규칙
- 썰툰 1~2컷은 **블라인드 UI 그대로 전면 재현**. 실제 앱 캡처처럼 보일 정도로 충실히.
- UI 컷은 **차가운 블루톤**. 본편(회상·현실) 과 톤이 대비되도록.
- 펀치라인(댓글 답글)이 뜨는 순간을 **UI 안에서** 처리하면 개그가 산다.
